# QA 스모크 시나리오 (`docs/qa-smoke.md`)

> 소유자가 배포본에서 반복 수행하는 스모크 체크리스트. 2026-07-12 첫 스모크를 승격한 것.
> 각 릴리스(웨이브 게이트 후 배포)마다 전 항목을 돈다. 실패 항목은 증상 그대로 총괄에게 전달.

## 0. 사전 조건

- [ ] `PUBLIC_WORKER_URL` 저장소 변수 존재 + Deploy Web green
- [ ] Worker healthz 200 (`/healthz`)
- [ ] Cloudflare rate limit rule 활성 (쓰기 경로 — `starts_with(path,"/g") and method in {POST,PUT}`, 2회/10초)

## 1. 열람·재생 (업로드 없음)

- [ ] .ttrm **드롭**으로 열기 (대용량 ~6MB 포함 — 2026-07-12 결함 회귀 확인)
- [ ] .ttrm 파일 선택 다이얼로그로 열기
- [ ] .ttr(솔로) 열기 — **미노 순서가 실제 경기와 일치** (2026-07-12: seed 1397564605 불일치 결함)
- [ ] 라운드·플레이어 선택 — **플레이어 인덱스가 보드 기준** (승패 순 아님 — 2026-07-12 결함)
- [ ] 재생·일시정지·스크러버 seek·배속(0.25×~4×)·프레임 스텝

## 2. 리플레이 업로드

- [ ] **업로드 버튼이 보인다** (2026-07-12: 부재 결함)
- [ ] 라운드 다중 선택 + 용량 표시 → 업로드 → `?gist=` URL 전환·복사
- [ ] 부계정 대시보드에 `[tetorial]` secret gist 생성 확인

## 3. 시뮬레이터·노트

- [ ] 일시정지 → 분기 → 시뮬레이터 진입
- [ ] **키 조작: DAS/ARR/SDF가 홀드 중 연속 동작** (2026-07-12: tick 미배선 증상)
- [ ] **홀드·넥스트 표시** (2026-07-12: 미표시 결함)
- [ ] **그리기 셀이 커서 위치에 정확히** (지우개·하이라이트 포함 — 2026-07-12: 오프셋 결함)
- [ ] 페이지 추가·주석·불러오기·삭제, undo/redo (**버튼 클릭 후 Space가 조작으로 새지 않음** — 포커스 결함)
- [ ] **시뮬레이터 종료 시 분기 프레임으로 복귀** (2026-07-12: 첫 프레임 복귀 결함)
- [ ] 노트 업로드 → 사이드바 즉시 갱신
- [ ] 다른 브라우저(시크릿)에서 공유 URL 열람 + 페이지 딥링크 복사·접속 (`note=<clientId>.<noteId>` 형식)

## 4. 편집 키

- [ ] 원 브라우저 "이어서 편집" 동작
- [ ] 시크릿 창에서 편집 시도 → 403 문구

## 5. 방어·오류

- [ ] `WRITE_ENABLED=false` → 저장 중지 문구 → 복원 (선택)
- [ ] 존재하지 않는 gist URL → "찾을 수 없습니다" 문구
