## 2. 규범(規範) — 재밀착 불변식

- **불변식 정의**: ARR 0이고 DAS 충전 완료 상태로 방향을 유지 중이면, 미노는 항상 그 방향 벽에 밀착해 있어야 한다. SDF ∞(또는 `sdfMs ≤ 0` floor mode)로 소프트드롭 유지 중이면, 미노는 항상 바닥에 밀착해 있어야 한다. 1회 이벤트가 아니라 **유지 상태의 불변식**이다.
- **재적용 트리거**: 엔진 상태를 변이시킬 수 있는 디스패치(회전 3종·swapHold·hardDrop의 락 후 새 미노·이동·소프트드롭 계열) 직후 재적용한다. 엔진은 무중력이므로 디스패치 밖에서는 상태가 변하지 않는다 → tick 시점 재적용은 불요.
- **순서**: 상태 변화 → 수평(`moveToWall`) → 수직(`softDropToFloor`). 한 축의 재적용이 다른 축을 다시 가능하게 하면(예: 소프트드롭으로 턱 아래 진입 → 수평 여지 발생) 두 이동이 모두 실패할 때까지 반복한다(고정점). 각 반복은 미노를 실제로 전진시키므로 보드 크기에 유계 — 무한 루프 없음.
  - ※ 순서는 **잠정 규범**이다. 게이트 11에서 소유자가 TETR.IO 실동작과 대조하며, 다르면 실측이 이긴다.
- **가드**: `currentPiece === null`이면 재적용 없음. DAS 미충전(`t < dasStart + das`)이면 수평 재적용 없음 — 불변식은 충전 후에만 성립. suspend 중 무동작은 기존 유지.
- 결정론(I-6) 유지: 재적용은 디스패치 순서에서만 파생되므로 같은 (press/release/tick, t) 열 → 같은 엔진 호출 열이 성립해야 한다.

## 수용 기준 (작업 세션 완료 조건)

- **I-7 수평 재밀착 불변식**: ARR 0·DAS 충전 유지 중 회전(킥 포함)·홀드로 미노가 벽에서 떨어지면 그 디스패치 직후 즉시 벽 재밀착 (#41 재현 시나리오: 좌홀드 + I미노 회전 → 회전과 동시에 왼쪽 벽)
- **I-8 락 후 재밀착**: ARR 0·방향 홀드 중 하드드롭 → 새 미노 스폰 직후 즉시 벽으로 이동 (기존 SDF ∞ 재적용 패턴과 대칭)
- **I-9 수직 재밀착 불변식**: SDF ∞(floor mode) 홀드 중 회전으로 부양하면 직후 즉시 바닥 재밀착
- **I-10 기존 경로 무변화**: ARR > 0·SDF 유한 반복 경로는 동작 불변 — 회전 후 즉시가 아니라 다음 ARR 틱에 이동함을 테스트로 고정 (구 I-1 "moveToWall 1회뿐" 테스트는 새 의미론으로 개정)
